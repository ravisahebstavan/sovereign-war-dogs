▸ TECH STACK

Built With

Every component chosen for performance and zero cost.

Rust + Tokio
Sub-millisecond event ingestion. Async WebSocket server on axum. Crash-resistant supervisor loops.
FinBERT (ProsusAI)
Financial sentiment model. Scores every headline −1.0 to +1.0 across positive/negative/neutral.
spaCy NER
Named entity recognition to extract company names and map them to ticker symbols.
Redis Streams
sovereign:news · sovereign:contracts · sovereign:events. Persistent, ordered, fan-out message bus.
USASpending.gov
Real DoD contract awards. No API key. Up to 3× signal boost on large defence contracts.
Tauri + React 18
Native desktop app. Live signal feed, intelligence matrix, alpha distribution — all in a dark UI.
